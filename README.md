# 🤖 Machine Learning Projects Portfolio

A collection of 5 end-to-end Machine Learning projects
covering Classification, Regression, Clustering,
and Anomaly Detection.

## 📊 Projects

### Project 1 — Breast Cancer Detection
- **Type:** Binary Classification
- **Algorithm:** KNN (tuned with GridSearchCV)
- **Result:** Recall = 100% ✅
- **Key Skills:** Pipeline, Cross Validation, Medical ML

### Project 2 — House Price Prediction
- **Type:** Regression
- **Algorithm:** XGBoost (tuned with GridSearchCV)
- **Result:** R² = 84.33% ✅
- **Key Skills:** Feature Engineering, Regression Metrics

### Project 3 — Customer Churn Prediction
- **Type:** Imbalanced Classification
- **Algorithm:** Logistic Regression + SMOTE
- **Result:** Recall = 79.14% ✅
- **Key Skills:** SMOTE, Imbalanced Data, Business ML

### Project 4 — Customer Segmentation
- **Type:** Unsupervised Clustering
- **Algorithm:** K-Means + PCA
- **Result:** 5 meaningful customer segments ✅
- **Key Skills:** Elbow Method, Silhouette Score, PCA

### Project 5 — Credit Card Fraud Detection
- **Type:** Anomaly Detection
- **Algorithm:** Logistic Regression + SMOTE
- **Result:** Recall = 100% ✅
- **Key Skills:** Fraud Detection, Extreme Imbalance

## 🛠️ Technologies Used
- Python
- Scikit-learn
- XGBoost
- Pandas, NumPy
- Matplotlib, Seaborn
- imbalanced-learn (SMOTE)

## 📈 Skills Demonstrated
- Data Cleaning and Feature Engineering
- sklearn Pipeline (no data leakage)
- Cross Validation (StratifiedKFold, KFold)
- Hyperparameter Tuning (GridSearchCV)
- Model Evaluation (Accuracy, Recall, F1, ROC-AUC, R²)
- Handling Imbalanced Datasets (SMOTE)
- Unsupervised Learning (K-Means, PCA)
